# JWT(JSON WEB TOKEN): 
主要的功能是安全性，當你第一次登入API要輸入帳號密碼，然後就會配發給你一個TOKEN。
當你自動登陸，是看cookie跟session存取了帳號密碼或token，若是存取token相對安全，因為當token遺失到別人手上，只要讓token失效，他就無法再登錄。
